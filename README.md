# Hue

Hue is an alternative communication tool for people who struggle to put their emotions into words. 

By making three selections: color, intensity, and energy levels, Hue is able to remove the need for words when communicating emotions and help users build a greater self-awareness over time.

## Features

- Three-step emotional check-in flow
- Interactive color wheel
- Intensity and energy sliders
- LocalStorage check-in saving
- Reflection summary dashboard
- Recent check-in color palette
- Emotional map visualization
- Selected check-in details
- Responsive mobile-first design

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
